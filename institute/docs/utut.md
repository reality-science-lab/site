## 0. 表記

正式表記は **`UTUTU`（5文字）**。

「現実」の訓読み「うつつ（u-tsu-tsu）」の **u** と **tsu→T** を交互に並べた表現で、`U-T-U-T-U` の5文字。MTG 文字起こしで一部「UTUT」と聞こえる箇所があるが、ロゴ・提案資料はすべて UTUTU。

---

## 1. 由来

| 要素 | 内容 |
| --- | --- |
| 語源 | **「現実」の訓読み =「うつつ」** |
| 文字構成 | **U・T・U・T・U**（5文字で「うつつ」） |
| 読み | ウトゥトゥ／ユーティーユーティーユー |
| キャラ名 | **うつつくん**（U君と T君の集合） |
| モチーフ | 鳥（複数色の鳥が U と T のかたちを構成する案／文字記号のみの案 両系統あり） |

「現実は、編集できる」という研究所のコアメッセージを、訓読み「うつつ」を介してビジュアル言語に翻訳した記号。

> 39:24 Keiko：「うつつっていうのをどっかに入れたい」
> 39:52 Keiko：「構成する2つは U と T にして、うつつを UTUTU にしたい」
> 43:20 chiaki：「いいっすね。うつつくん。」

## 2. コンセプトの4軸

UTUTU は1つの記号で4つの意味を同時に運ぶ：

| 軸 | かたち | 表現すること |
| --- | --- | --- |
| **レイヤー性** | 文字同士が重なって違う色に見える／3色のアクセントが U の凹みに入る | 現実は1つではない／編集できる |
| **凹凸（おうとつ）** | U＝凹 ／ T＝凸 | 見る側と見せる側、内と外の対称 |
| **編む（ウィービング）** | U と T が互い違いに並ぶと布の拡大に見える | 「文章を編む」「品種を編む」と編集の同型 |
| **目（まなざし）** | 鳥の目／文字の中の色面 | 「どう見るか／どう見せるか」＝杉山（脳科学：見るの側）と藤井（XR：見せるの側）の継承構造 |

> 41:00 chiaki：「うつつたちが重なり合って違う色になったり、違う見え方を見せてくれる」
> 45:09 鈴木：「U と T が互い違いに並ぶと布の拡大みたいにも見える、編集の編むに近い」
> 51:08 Keiko：「目は大事。どう見るのか、どう見せるのかという点で」

## 3. UTUTU が解決する課題

これまでのビジュアル方針（赤＋真っ黒の高コントラスト）に対する違和感を、UTUTU が引き受ける：

- **都市伝説感の払拭**（鈴木 43:36「都市伝説感がある」／chiaki 43:47「直木マンショー」「なんとかドグマ」感）
- **ケバケバ感の中和**（プロクロ背景・原色配色の緩和）
- **昭和感／軍事感を削る**（46:46 Keiko）

> 50:51 chiaki：「**このうつつくんの登場によって、多分都市伝説は消えていくのではないかと思っている**」

## 4. ビジュアル素材バリエーション（PDF 採用ページ）

提案 PDF（`assets/utut/UTUTU_v1_2026-06-05.pdf`）の **採用ページのみ** を用途別に整理。
棄却ページ（p.1 / p.4 / p.5 / p.6）は §4.7「棄却案と理由」に分離して記録。

### 4.1 公式ロゴ系（p.8）— 最優先

- **漢字「現実」をUTUTU 字形で組んだロゴ**：「現実 / Reality Science Institute」（縦書き Reality Science Institute 添え）
- **シンプルロゴ**：「UTUTU / Reality Science Institute」
- 背景：ベージュ系（教科書紙の質感）／文字：白〜濃グレー

→ **公式ロゴ最有力候補**。ベージュ＋濃グレー＋白の禁欲的なパレットが MTG 決定の「昭和感・軍事感を削る」方針と合致。Hero／Footer／印刷物の正面で使う。

### 4.2 透過レイヤー（p.2）

- 透過40 / 透過60 / バラバラ / 透過なし
- 黒〜濃グレー背景の上に UTUTU を異なる透過率で重ねる
- **「バラバラ」版**：透過率違いの3つの UTUTU を縦に重ねた状態。「複数の現実が同時に見える」表現

→ 「現実は層である／編集できる」のコアメッセージを最も直接的に示すモーション素材。スクロールアニメ・ヒーロー展開で活用。

### 4.3 縄文紋様背景（p.3）

| # | パターン | 用途 |
| --- | --- | --- |
| **縄文 土色** | 縄文土器の渦巻き紋様 ＋ 白 UTUTU | 日本古来の「現実観」の参照／土の温度感 |
| **縄文 白黒 ライン** | 線画の縄文紋様 ＋ 濃グレー UTUTU | ペン画風・印刷向き |

→ 「和」軸はこの **縄文一本** に統合（障子は棄却）。「うつつ」という大和言葉と、紋様が編むように渦を描く視覚が、UTUTU の「ウィービング」軸と二重に響く。

### 4.4 キャラクター・素材サンプル（p.7）

- **手描き顔つき UTUTU**（左上）：U と T それぞれに顔。「うつつくん」のキャラ展開
- **鳥イラスト**（右上、加賀さん画）：U・T 形に近い鳥の集団。マスタード地に黒鳥
- **カラフル積み重ね**（左下）：青・緑・ピンク・黄の U/T が積み重なって UTUTU の壁を作る
- **ピンク・ティール対比**（右下）：マゼンタとティールの2色で UTUTU を構成

→ **限定的に展開**：公式ロゴは無表情・無彩色。キャラ展開（顔つき・カラフル）は SNS／教育／物販シーン専用とする。LP 本体では使わない。

### 4.5 組み合わせパターン（p.9-12）

- **組み合わせイメージ2**（p.9）：U の中に T が入る／オレンジ・ピンク・ベージュ・マスタードの面塗り
- **バラバラ／組み合わせ 黒（p.10）／白（p.11）／グレー60%（p.12）**：単色シルエットの並び・組み方バリエーション

→ ロゴシステムとして「文字を分解して再構成できる」可塑性が確保されている。LP のセクション切り替えや配信物のテンプレートで活用可能。バラバラ＝編集行為の視覚化として、p.2 の透過レイヤーと組み合わせると強い。

### 4.6 ロゴシステムの3階層まとめ

| 階層 | 使用素材 | 用途 |
| --- | --- | --- |
| **公式 (Formal)** | p.8 ベージュ漢字／シンプル | Hero、Footer、印刷物、学長室プレゼン、登壇資料 |
| **構造 (Structural)** | p.2 透過、p.3 縄文、p.9-12 バラバラ | LP セクション、背景パターン、装飾 |
| **キャラ (Character)** | p.7 顔つき・カラフル | SNS、教育、物販、コミュニティ |

階層を混ぜない。公式の場でキャラを出さない、SNS で公式ロゴだけにしない。

### 4.7 棄却案と理由（再提案防止のため記録）

| ページ | 案 | 棄却理由 |
| --- | --- | --- |
| **p.1** | 満たす カラフル 黒／グレー（U の凹みに黄・ティール・ピンクの3色面） | 「色を足す＝満たす」のメタファーは明快だが、ポップ／kawaii 方向に傾く。RSI の禁欲性と衝突。**3色アクセント案は不採用** |
| **p.4** | 白文字 グレー背景／黒文字 白背景（無テクスチャのフラット版） | 紙の質感・温度がない。**ベージュ紙質感が前提**になったため、無背景フラット版は使わない |
| **p.5** | グリッチ背景（白／黒、人物入り） | デジタル揺らぎ系。情報量過多で、「昭和感削る／落ち着き」方針と矛盾 |
| **p.6** | 障子背景 | 和素材として直接的すぎる（ステレオタイプ）。縄文（p.3）で和は十分カバー |

→ これらの方向で再提案が来た場合、本ドキュメントを参照して棄却理由を確認すること。

## 5. カラー方針（MTG 決定＋PDF 採用ページ）

p.1 のカラフル3色案を棄却した結果、**ベージュ＋濃グレー＋縄文土色** の禁欲的パレットに収束：

| 項目 | 変更前 | 変更後 |
| --- | --- | --- |
| 基本黒 | 純黒 `#000` | **濃いグレー** |
| 背景ベース | 白／プロクロ | **ベージュ＋紙のザラザラ感**（p.8 公式ロゴ案、p.3 縄文 土色） |
| アクセント色 | 朱色 1点 | **公式は無彩色**。朱は LP の「編集の赤」として継続。キャラ展開時のみカラフル可（p.7 限定） |
| プロクロ背景 | 主役級 | 控えめに（ケバケバ感の原因） |

> 50:11 Keiko：「黒を多分濃いグレーにするだけで」
> 49:42 Keiko：「教科書材とかなんか紙感があっていい」「もしかしたらさらに紙っぽくザラザラ感、手触り感を出してみるのもあり」

### 5.1 暫定カラートークン案（要 Keiko 最終承認）

```
/* 公式パレット（LP本体・印刷物・登壇資料すべてここ） */

--utut-jomon-earth:  #6B4A2E  /* 縄文土色（p.3 上、土器の地色） */
--utut-edit-red:     #D6422E  /* 編集の赤（既存 design.md から継続。1スクリーン1点） */
--utut-paper-grain:  url(...) /* 紙テクスチャ（ザラザラ感） */

/* キャラ展開専用（SNS/教育/物販。LP本体では使わない） */
```
