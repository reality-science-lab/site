## 世界を満たすプロジェクトを、動かす場へ。

### ——「デジハリラボ」、在野・起業家・国の力を受け止めるプラットフォームとして始動

デジタルハリウッド大学（学長：藤井直敬）は、創立者の杉山知之、政策・教育の中村伊知哉と共に、新しいプラットフォーム「**デジハリラボ**（仮称）」を立ち上げます。すでに動き始めている**5つのプロジェクトを起点**に、**在野の研究家、起業家、国——それぞれの力をひとつに受け止め、社会を動かしていく場**として始動します。

---

## なぜ、いま、このラボなのか

日本には、領域や組織の枠を越えて社会を動かそうとする「大きな問い」を持った人々が、各所で個別に動いています。アカデミアの内外、産業界、行政、そして肩書きの外側で問いを立て続ける「**在野の研究家**」たち。

しかし、それぞれの力を横でつなげ、社会の規模で**ひとつのプロジェクトとして動かす場**は、日本にはほとんどありません。複数の領域・組織の力が交差しなければ動かせない問いは、既存のアカデミアにも、企業単独にも、行政だけにも収まりきらない。

デジハリラボは、**そのような問いを動かすプロジェクトを起点に、在野・起業家・国の力を受け止めるプラットフォーム**として始動します。

---

## 動いているプロジェクト

デジハリラボは、すでに動き始めている5つのプロジェクトを起点に始動します。

**いずれも、ひとつの組織や領域では動かしきれない問いを、産・官・学・在野の力を横断させて動かしているプロジェクトです。**ポップカルチャー、XR、教育の構造、シンギュラリティ、ナショナルセキュリティ——テーマは多様ですが、共通しているのは「**既存の制度の中だけでは動かない**」「**意思を持つ主格者が、組織や仲間、コミュニティを率いて動かしている**」「**領域・世代・国境のいずれかを越境している**」こと。

ラボはこのキュレーションを軸に、これからも参画するプロジェクトを選び、動かしていきます。

### Project 01｜PPP（Pop Power Project）

**ポップカルチャーを横断する専門家コミュニティ**

マンガ・アニメ・ゲーム・音楽など、日本の各ジャンルに横串を通し、テクノロジー融合・海外市場開拓・他産業連携を通じてポップカルチャー全体の活性化と国際化を図る、産官学横断のプロジェクト。中村伊知哉氏（iU学長／CiP協議会理事長）を中心に、講談社、SEGA、コナミ、エイベックス、TBS、円谷プロ、TMSエンタテインメント、内閣府知的財産戦略推進事務局、経済産業省、文化庁など、多数の企業・省庁・業界団体が結集。本学からも複数の教員が個人会員として参画してきた本プロジェクトを、デジハリラボに移管し、本学を共同事務局拠点のひとつとしてさらに推進していきます。

### Project 02｜XRコンソーシアム

**日本のXR業界（VR/AR/MR）の代表団体**

藤井直敬が2015年に発起し、代表理事を務めてきた一般社団法人。落合陽一氏、せきぐちあいみ氏ら理事陣とともに、XRクリエイティブアワードを10年以上にわたり開催し、日本のXRシーンを牽引してきました。近年はAI×XRの交差点へとテーマを拡張し、「AI REALITY ― 会いましょう、AIとXRの交差点で」をコンセプトに、AIコンテンツワーキンググループ勉強会の継続開催、海外・関係省庁への業界窓口、ガイドライン策定など、強固なエコシステム構築を推進。デジハリラボの基盤と接続し、大学の研究・教育リソースとXR業界・AI業界をより深く結びつけます。

### Project 03｜雨ニモマケズ高等学校 連携

**「人生、急がないための学校」との高大接続**

2027年4月開校予定の広域通信制・単位制普通科 雨ニモマケズ高等学校（校長：大﨑洋氏／吉本興業ホールディングス前代表取締役会長、設置法人：学校法人 幸和学園）と本学は、令和8年4月1日付の連携ご依頼を受け、産学官連携フレーム「メディアラボ」を基盤に、田中悠斗研究室と高校が連携参画する新しい高大接続モデルを構築します。AI・サイバーなど「最先端の学び」と、農業・自然体験・サバイバルなど現実と向き合う「最遅端の学び」を往還する統合的な教育実践により、高校段階から大学研究、さらには社会実装へと連続する学びの導線をつくります。

### Project 04｜シンギュラボ連携

**シンギュラリティに適応する「新しい人類」を共に育てる**

株式会社スペースデータ（代表取締役社長：佐藤航陽氏）が運営する、シンギュラリティ（技術的特異点）に適応した「新しい人類」を目指す研究コミュニティ「シンギュラボ（Singulab）」と、本学は2026年4月に戦略的コラボレーションを開始しました。藤井学長自身がシンギュラボの専門家・アドバイザーを務める縁を起点に、宇宙・AI・ロボティクス・仮想現実・脳科学・IPなど多領域の知性を統合・実験する場として、(1) 実践型コラボ勉強会、(2) 共創プロジェクトの立上げ、(3) 「シンギュラボ・アジト」の教育現場への拡張、の3本柱で本学のクリエイティビティとシンギュラボの実装力を融合し、未来の教育環境を共に構築します。

### Project 05｜サイバーセキュリティ田中プロジェクト

**ナショナル・セキュリティ教育研究の新拠点**

ホワイトハッカーとして著名な本学教授・田中悠斗（株式会社フォアー）を中核に、デジタルハリウッド大学を拠点とするナショナル・セキュリティの教育・研究を推進するプロジェクト。データサイエンス／サイバーセキュリティ／分散型台帳技術を軸とするディープテックの知見と、ベトナム・Duy Tan大学との国際連携、CYBERCON ASIAなど国際的な実装フィールドを背景に、本学のクリエイティブ教育と接続。雨ニモマケズ高等学校との高大連携を通じて、高校段階から最先端のサイバー領域に触れることのできる、これまでにない教育・研究エコシステムを構築します。

今後、参画するプロジェクト・コミュニティを順次発表します。

## ラボがプロジェクトを支える3つの基盤

ラボはプロジェクトを「動かす場」として機能しますが、立ち上げたプロジェクト・研究・コミュニティを中央に吸い上げることはしません。**それぞれの問いと活動は、メンバーが自分のものとして持ち続けます。**ラボは、その独立性を支える3つの基盤を提供します。

### 1. 攻める研究を可能にする「株式会社立大学」という器

デジタルハリウッド大学は、2005年の開学以来、**株式会社立大学**という日本でも数少ない形態で運営されてきました。だからこそ、国立大学では制約がかかる領域——たとえば安全保障や、既存の学術倫理の枠を越境する実験的研究——にも、アカデミアの正統性を保ったまま取り組むことができます。

20年にわたりコンテンツ・テクノロジー産業と隣り合わせで走り続けてきた実績が、プロジェクトと社会の距離を縮める基盤になります。

### 2. 30年が育てたネットワークを、社会への回路に

- **5,000人を超える卒業生**が、コンテンツ・テクノロジー業界の現場で活動しています
- **500人を超える元教員・現教員**が、研究と実践の両側で動いています
- **全国50箇所以上のデジハリスタジオ**が、地方の挑戦と都市部の知をつなぎます

産業・行政・国際の各方面に伸びてきたこのネットワークを、プロジェクトを社会に届けるための回路として活用していきます。

### 3. 力を受け止めるプラットフォーム機能

在野の研究家、起業家、国——それぞれの力は別々の岸にあります。ラボは、その間に渡す橋として、また、それぞれの力が交わる結節点として機能します。コミュニティが個々に抱え込んでいた事務的・基盤的な負担をプラットフォーム側で引き受けていくことで、メンバーが「自分の問い」と「他者と語り合うこと」に集中できる環境を目指します。

## 【内部メモ】v0.2 の設計判断（外には出さない）

> このセクションは社内共有用。プレスリリース本文には含めない。

### v0.1.3.5 → v0.2 の構造転換

主軸を「**在野の研究家のハブ**」→「**プロジェクトを動かし、在野・起業家・国の力を受け止めるプラットフォーム**」に転換した。

転換の3つの起点：

1. **藤井指摘（2026-05-08）**: 「在野の研究者は大きな流れを作る力がない／在野で見えない存在。多くの人が参画できるプロジェクトを動かすことを起点にしたらどうか。在野・起業家・国の力を受け止めるプラットフォーム」
2. **太田前回コメント**: 「外に出すのはコンテンツだけでいい」→ フィロソフィー声明を外で立てるのではなく、選ばれたコンテンツ（プロジェクト）に語らせる構造へ
3. **Chiaki方針**: 「魅力的なプロジェクト群」だけだとぼやけるので、**キュレーション原則の可視化**で尖りを立てる（テーマ声明は内部資料に逃がす）

### 採用した構造的判断

| 判断 | 理由 |
|---|---|
| 主見出しを「研究を、民主化する。」→「世界を満たすプロジェクトを、動かす場へ。」 | 「動かす」「受け止める」（藤井語彙）と「Fill the World」（学長ミッション）を一文に同居 |
| 「在野の研究家」を主役 → 3層（在野・起業家・国）の1つに格下げ | 在野は重力源にならない（藤井指摘）。プラットフォームが3層を受け止める構造へ |
| 「初期に走るプロジェクト」を後半 → 前半に移動、見出しも「**動いているプロジェクト**」へ | プロジェクト起点の構造を実装。コンテンツが先、構造説明は後 |
| プロジェクト導入文に**キュレーション原則の一段落**を追加 | 「いいものを集めた」→「**選び方の原則がある**」へ。尖りの実装ポイント |
| 旧「何をやるのか」3節 → 「ラボがプロジェクトを支える3つの基盤」に再構成 | 主役はプロジェクト、ラボは支援構造、という関係を明示 |
| 「3つの基盤」の順序：① 攻める研究 ② 30年ネットワーク ③ プラットフォーム機能 | 既存資産（株式会社立／30年）を先に出して信頼性を担保し、③で藤井の「受け止める」を受ける |

### キュレーション原則（外向き一文／内部NOT-list）

**外向き（プロジェクト導入文に明示）**：
> 「ひとつの組織や領域では動かしきれない問いを、産・官・学・在野の力を横断させて動かしているプロジェクト」

**内部の採択基準**：

✅ 採択する
- 既存のアカデミアでは扱いきれないテーマ
- 公的または産業的なスケールを持つ
- 領域・世代・国境のいずれかを越境している
- 意思を持つ主格者が、組織やコミュニティを率いて動かしている

❌ 採択しない
- 単一領域に閉じた基礎研究
- 個人スケールの探究プロジェクト
- 既存の学術制度で十分扱える研究
- 社会接続のないコミュニティ運営

### 引き続き出さないこと（v0.1.3.x から維持）

- サーバーラック / 業界団体インキュベーション
- 分科会・コンソーシアム構造
- MITメディアラボ超越宣言
- 収益還流サイクル
- 大量集客の目的化
- 「問いの品種改良場」（内向き構造説明、LP寄りで使う）
- 助成金分配スキーム等の運営メカニクス

### v0.2 で外向きに出さないが、内部資料として彫るべきもの

- **テーマ声明（内部用）**: 「ラボとは何の動詞をする組織か」を一文で。チーム・参画者向けの軸として
- **採択原則とNOT-list**: 上記。プロジェクト追加判断時の揺れを防ぐ
- **資本分散型モデルの運営詳細**: 助成金分配・リターン構造・評価軸など
- **「袋閉じ」概念**: v0.1.3.x で表現として使っていたが、v0.2 では外さず、内部での方向性表現として保持
