## v3 との関係

v5 は v3 のフルコピー＋以下の確定状態を含む:

1. 趣意書 v2 (現実科学研究所 設立趣意書 平易版) の本文反映
2. プロジェクト順序を「文化 → 創発 → 境界 → 自己」(PPP → シンギュラボ → 能動的セキュリティ学 → 雨にも負けず高校) に
3. Hero 主役を「現実科学研究所」(漢字7文字) に、UTUTU は背景透かしのみ
4. Hero subtitle ローテーター完全停止 (manifest 内 JS の `applyCatch` / `setInterval` を patch)
5. 「現実は、編集できる。」固定
6. Edit ボタン削除、縄文背景削除、`U・T・U・T・U / うつつ ＝ 現実` 削除、bottom strip 削除
7. Footer の UTUTU ロゴ非表示、メイン文字列を「現実科学研究所」に
8. Footer の `EST. 2026.07` ベース表示 (旧「・ 趣意書 V.2」を削除)
9. vcard h3 (プロフェッショナルたれ等) の CJK 中折り対策
10. Hero 全体を初回ビューポート内に収まるよう `height: calc(100svh - 24px)` 強制

## v3 / v5 を分けた理由

- v3 は「LP 本番候補の sandbox」として実験的に編集を重ねる場
- v5 は「クライアントへ共有する版」として、編集の節目ごとに固定スナップショットを切り出す

v3 を引き続き編集する場合も、v5 のリンクはこの時点の表示で固定される。
