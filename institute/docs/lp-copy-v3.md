# v3 LP — テキスト編集ファイル

**使い方:**

1. `###` の下の本文を編集する（`###` 自体は触らない）  
2. 編集完了後、Claude に「v3 反映して」と伝える  
3. または `python3 docs/lp-copy-v3-apply.py` を実行

**ルール:**

- 改行は普通に書ける（`<br>` 自動挿入）  
- 空行で値の終わりとする（次の `###` まで継続）  
- HTML タグはそのまま書ける（プレーンテキスト扱い）  
- 太字・赤強調等のスタイルは構造に紐づく — テキストだけ書き換え
