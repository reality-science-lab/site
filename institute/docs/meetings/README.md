## 批判的検証レポート — 2026-04-15

**形式:** 3ラウンド・9チームによる外部分析統合レポート（藤井学長たたき台への批判検証）
**レポート:** [[2026-04-15_批判的検証レポート]]
**原本:** `~/Projects/_knowledge/inbox/2026-04-15_デジハリラボ構想_批判的検証と提言.md`

### 主要発見

| 主張（たたき台） | 検証結果 |
| --- | --- |
| コンテンツ産業の人脈密度 | 意思決定層に卒業生がいる証拠なし。アニメ＝美大、ゲーム大手＝四年制 |
| 株式会社立だから事業化しやすい | 制度的には正しいが、ベネッセ傘下化との矛盾 |
| 理研コンツェルン型 | 国策保護・長期金融の条件が現代に再現不可 |
| MITメディアラボを超える | 大言壮語。エプスタイン問題を抱えたブランドリスク |
| 収益還流サイクル | **世界13事例で完全実現はゼロ**。Stanfordすら還流率16%。10-20年後のゴール |
| コンテンツ産業のR&D需要 | 大手は自前。中小は研究知見ではなく制作費・採用チャネルを欲する |

### 提案された新ビジョン

> **「問いの品種改良場」** — 答えの生産工場ではなく、問いの生成・検証・淘汰を高速で回す。問いの寿命6-18ヶ月。

- デジハリのDNAは「**適応速度**」（人脈や理研型ではない）
- 杉山→藤井の継承は「拡張」から「現実の再設計」への問いのシフト
- 「3人ともMIT」は偶然ではなく、日本の構造的欠損の証拠

### 提案された3プロジェクト（参考案）

- コンテンツ効果測定ラボ（年2,000-3,000万）
- AI×XR倫理・品質基準策定（年1,000-2,000万）
- 現実科学×起業家育成プログラム（年1,800-4,000万）

→ **後の第3回MTGで採用された5プロジェクト（smplprj）とは別系統**。本提案は将来の研究テーマ候補として保持。

## 第2回 — 2026-04-17（構想と動機の深掘り）

**出席者:** 加藤 / 藤井直敬 / 太田 / 他1名 ／ 56分
**議事録:** [[2026-04-17_第2回準備MTG]] ／ **トランスクリプト:** [[transcripts/2026-04-17_第2回_transcript]]
**原本:** `~/Projects/_knowledge/log_meeting/2026-04-17_デジハリラボ.md`

### 主要決定

- ラボのモデルは**資本分散型**（売上は外、助成金をシードに新規事業）
- 柱は**佐藤和明・スペースデータの2社**＋市谷さんが連れてくる数名
- **3ヶ月で内容を詰めて社長提案**（社内承認プロセス）
- 「デジハリ」名称は社内ネガ反応あり、別名検討

### 主要論点（why-what.md の主要ソース）

- **藤井の根源的動機**: 「やらないと退屈」＋研究者時代の20-25億円使用への反省
- **アカデミアの構造的無駄遣い**: 100人中99人が誰にも知られず終わる
- **ラベリングしない個性**: ラボ自体が「定義できないものとして始まる」と藤井
- **二層構造**: 内側は定義しない／外側に「在野の科学者・研究家」のような新ジャンルを作る
- **三越的な人集め力**: 中村（市谷）が説明コストの低い場を作る
- **共依存的役割分担**: 藤井＝開拓者、太田＝楽しめることをやる実行者
- **MIT初対面**: 藤井と中村は2000年頃にMITで出会った（蝶ネクタイの中村）

### この回で出てきた重要概念

- 「**在野の科学者・研究家**」
- 「**Fill the World of Yours（あなたの世界を満たせ）**」の前段（実際の言語化は第3回で）

→ この回の整理が [[why-what.md]] にまとまっている。

## 第3回 — 2026年5月初旬（構造と発信方針の確定）

**出席者:** 加藤 / 藤井直敬 / 太田 / 小田原 等
**解析:** [[2026-05_第3回解析]] ／ **トランスクリプト:** [[transcripts/2026-05_第3回_transcript]]

### 主要決定

1. **ラボの場の機能を「サーバーラック / 業界団体インキュベーション」モデルとして言語化**
   - インフラ提供に徹する（事務局・法人化コスト肩代わり）
   - 比喩: 「0050」「アイブリー」「AWSのコンソーシアム版」
   - 月1-10万で乗れる業界団体ラック
2. **外向きメッセージは「研究を、民主化する」一本に絞る**
   - 「新しい産業を作る」は内部動機として保持、外には出さない
   - 「外には富士山だけ、サーバーラックは見せない」原則
3. **ターゲットの絞り込み**: 「**袋閉じ**」の領域＝まだラベルが貼られていない人（Discord文化、Roblox/Minecraft異能、在野の研究家）
4. **やらないことの追加**:
   - うぞうむぞの大量集客はしない
   - 中身の運営はしない（インフラに徹する）
5. **「あなたの世界を満たせ／ Fill the World of Yours」**: ラボ拡張案として議論。**藤井学長との合意までは内部のみ**

### 周辺確認事項

- 学長交代: **藤井直敬が2026-04-01付で就任**（杉山は創立者として継続）
- 大学全体ミッション: **「世界を満たせ／ Fill the World」**（2026-03-15 学位授与式で初公開）
- 藤井学長note記事に**「MediaLab」**（2026-04始動・梁山泊・世界を満たしていく場所）の記述あり
- 詳細: [[fill-the-world.md]]

### 直近のアクション

- **2026-05-06（水）まで** に Chiaki が中間アウトプット → [[press-release-v0.1.md]] として完成
- **2026-05-08（金）17:00-18:00** に第4回MTG
