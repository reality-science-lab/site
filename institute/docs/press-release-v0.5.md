## 現実は、編集できる。

### ——デジタルハリウッド大学「現実科学研究所」設立。学長 藤井直敬が掲げる「世界を満たせ」を社会に実装する拠点として始動

デジタルハリウッド大学（学長：藤井直敬）は、2026年7月、**現実科学研究所（Reality Science Institute）** を設立します。

本研究所は、藤井学長が提唱する「**現実科学**」を中核思想に据え、研究者・クリエイター・産業界・行政が同じテーブルで「**現実編集**」を実践する産官学連携拠点です。領域横断型の実装拠点を参照しつつ、日本のクリエイティブ産業と地続きの現場性を持つことを独自性とします。

立ち上げ時より **4つの旗艦プロジェクト** を起点に、本研究所のミッション「**世界を満たせ／ Fill the World**」を社会の規模で実装していきます。

## なぜ、いま——共同幻想が機能を失った時代の研究機関として

戦争、AIの急進、価値観の多様化——昨日まで信じていた「正しさ」が今日には通用しない時代に、私たちは生きています。万人共通の正しさが失われた今、多くの人が「自分一人に何ができるのか」という無力感を抱えています。

しかし、「答えがない時代」は、見方を変えれば「**自由な時代**」でもあります。誰かが決めた正解に従う必要がない。自分のやり方で、世界に新しい価値を加えることができる——本研究所は、その新しい知に「手を伸ばす／体験する／循環させる」場として設計されます。

デジタルハリウッド大学は1994年、「**すべてをエンタテインメントにせよ**」を掲げて創立されました。創立から30年余を経て、生成AI・XR・脳科学が「人間にしかできないこと」を書き換えつつある今、私たちは問い直します——この大学は、これから何を作り、何を学び、何を社会に届ける場なのか。

その答えとして、本研究所は設立されます。

## 思想的基盤——現実科学

### 現実は、私たちの内部に立ち上がる

脳科学が明らかにするように、私たちの脳は世界を客観的に受け取っているのではなく、過去の経験からつくった予測モデルをもとに「世界がどう見えるか」を能動的に組み立てています。だから、同じ場にいた二人が、まったく違う「現実」を生きていても不思議ではありません。

「現実は人によって違う」——この当たり前の事実を直視し、それを扱える知として体系化する試み。それが、藤井学長が提唱する **現実科学** です。

### 現実編集——哲学から方法論へ

「動かない」と思い込まれてきた前提や境界を、意識的に見直し、別のかたちへと組み直す実践——本研究所はこれを **現実編集** と呼びます。自分の生き方、チームのルール、会社の仕組み、社会の合意、文化のパッケージ、産業の構造——どのスケールでも、現実編集は起こりえます。

これまで一部の天才や名匠が直感で行ってきたこの編集を、**誰もが学べる技術へと引き上げる**——これが本研究所の挑戦です。

## ミッション「世界を満たせ／ Fill the World」——杉山時代からの継承と拡張

1994年、創立者・杉山知之が掲げた「すべてをエンタテインメントにせよ」——日常も教育もビジネスもテクノロジーも、すべて創造的で楽しい体験として再構築する。デジタルで人を解放し、感動を届ける——この精神は、デジタルハリウッドの30年の根幹でした。

2026年、藤井学長が新たに掲げる「**世界を満たせ／ Fill the World**」は、その精神を継承し、拡張するものです。エンタテインメントの領域を「**人間の可能性を拡張する知**」「**世界の見え方を編み直す力**」へと押し広げ、コンテンツ・サービスだけでなく、教育・医療・労働・ガバナンス・研究・外交——あらゆる領域で現実を編集し、世界を豊かさで満たしていきます。

第一の時代がデジタルクリエイティブ産業の血脈をつくったように、新しい時代は「**現実編集産業**」とも呼ぶべき新しい産業のエコシステムをつくっていきます。本研究所は、その継承と進化を同時に担う実装拠点です。

## なぜDHUか——研究者とクリエイターが同居する世界唯一の実装場

世界には優れた研究機関が多数存在します。MITメディアラボ、スタンフォード d.school、ETH Zürich、東京大学先端科学技術研究センター——それぞれが領域横断と社会実装を掲げています。

しかし、**研究者と、産業の現場で作品とサービスを生み続けるプロのクリエイターが、同じ屋根の下に常駐している場** は、世界に数えるほどしかありません。

デジタルハリウッド大学はその希少な場の一つです。設立以来30年余、研究者・教育者・実務家・アーティスト・起業家を同時に擁し、論文と作品の両方を世に送り出してきました。理論を語る人と、それを実際に形にする人が同じ場にいて、互いの言葉を翻訳し合えるとき、はじめて新しい考えが産業や社会に届きます。

現実科学研究所が、他の研究機関でも企業R&Dでもなく、デジタルハリウッド大学のなかに置かれる理由は、ここにあります。

## 4つの旗艦プロジェクト——現実編集の4スケール

本研究所は、立ち上げ時から取り組む **4つの旗艦プロジェクト** を通じて、「現実編集とは具体的に何か」を社会にショーケースします。それぞれが編集の対象スケールの異なる、現実編集の実例です。


| プロジェクト            | 編集スケール | 編集の方向 |
| ----------------- | ------ | ----- |
| 雨にも負けず高校連携        | 自己スケール | 内向き   |
| 攻撃学               | 境界スケール | 内向き   |
| シンギュラボ連携          | 創発スケール | 外向き   |
| Pop Power Project | 文化スケール | 外向き   |


### Project 01｜Pop Power Project（PPP）——文化スケールの現実編集

**ポップカルチャーを横断する専門家コミュニティ**

ポップカルチャー（アニメ・マンガ・ゲーム・音楽・キャラクター）は、産業規模で行われてきた現実編集の最も成功した例です。クリエイターが構築した主観的世界を、コンテンツというパッケージに実装し、世界に流通させる——日本のポップカルチャー全体の活性化と国際化を目的として2025年に発足したPop Power Project（PPP）の事務局を、本研究所が担います。

中村伊知哉氏（iU学長／CiP協議会理事長）を中心に、講談社、SEGA、コナミ、エイベックス、TBS、円谷プロ、TMSエンタテインメント、内閣府知的財産戦略推進事務局、経済産業省、文化庁など、多数の企業・省庁・業界団体が結集。本研究所はクリエイター・配給・テック・脳科学・哲学などの他分野の知を一つのテーブルに集め、コンテンツ産業を「現実編集産業」という視点で再定義します。

### Project 02｜シンギュラボ連携——創発スケールの現実編集

**シンギュラリティに適応する「新しい人類」を共に育てる**

株式会社スペースデータ（代表取締役社長：佐藤航陽氏）が運営する研究コミュニティ「シンギュラボ（Singulab）」と、本学は2026年4月に戦略的コラボレーションを開始しました。研究者・起業家・アーティスト・市民が境界を越えて集まり、ボトムアップで課題を発見し解決する創発的コミュニティと、本研究所が連携します。

トップダウンの「正解」を疑い、現場の個別主観から立ち上がる解——これは現実科学が描く「意識と無意識のあいだのダイナミクス」を社会規模で再現する試みです。共同幻想が崩れた時代に、「自分の現実を自分で定めて構築し続ければ、必ず外に伝わる」という現実科学の主張をコミュニティ規模で実証する場となります。

### Project 03｜攻撃学——境界スケールの現実編集

**ナショナル・セキュリティ研究の新地平**

本学・田中悠斗教授（株式会社フォアー）が中心となって立ち上げる「攻撃学」は、サイバーセキュリティ研究の新しい地平を拓くプロジェクトです。「**攻撃を知らずして防御は無理**」という命題は、現実科学の核心と完全に重なります。

防御だけでは予測モデルは硬直し、世界は固定化します。攻撃の視点に立つとは、自分が無自覚に守ってきた現実の輪郭を、外から見直すこと——「**現実を定義し直す勇気**」を技術として身につけることです。学長のSR（代替現実）研究が「現実は差し替え可能だ」と被験者に体感させた装置であったように、攻撃学は「あなたが守っているシステムは差し替え可能だ」と防御者に体感させる学問になります。

本研究所は、攻撃学を日本のサイバーセキュリティ研究開発のハブとして発展させると同時に、その方法論をセキュリティに留めず、組織・制度・自己定義の硬直を解く一般的な現実編集技術として横展開します。

### Project 04｜雨にも負けず高校連携——自己スケールの現実編集

**「人生、急がないための学校」との高大接続**

2027年4月開校予定の広域通信制・単位制普通科 雨にも負けず高等学校（校長：大﨑洋氏／吉本興業ホールディングス前代表取締役会長、設置法人：学校法人 幸和学園）のレジリエンス教育と、本学を高大連携で接続します。15〜18歳という、最も自己定義が硬化しやすい時期に、**可動域を保ったまま大人になる訓練** を提供する——これがこのプロジェクトの目的です。

レジリエンス教育を現実科学の視点で再定義すると、それは「逆境への耐性」ではなく「**自己定義を常時更新し続ける力**」になります。本連携は、現実科学を「大学院・研究の言葉」から「10代に届く生きる技術」へと翻訳する場となります。

今後、参画するプロジェクト・コミュニティを順次発表します。

## 研究所の4機能——研究・実装・教育・発信の循環

本研究所は、四つの機能を一つの循環として運営します。

- **研究**：現実科学を中核に、脳科学・XR・AI・ブレインテック・哲学・社会学・芸術理論を統合した学際研究。論文・特許・白書として知を生み出します。
- **実装**：研究成果と現場の課題を結びつけ、産業界・行政・市民社会との共同プロジェクトを進めます。プロトタイプ・サービス・コンテンツ・制度として、知を社会に届けます。
- **教育**：研究と実装の現場を、学部・大学院のカリキュラム、社会人向けリカレント講座、企業内研修、高大連携プログラムへと展開します。次世代の「現実編集者」を育てます。
- **発信**：知と実践を、出版・レクチャーシリーズ・国際カンファレンス・メディアを通じて開きます。コミュニティを形成し、共感者と協働者を世界中から集めます。

発信が共感者を集め、共感者が実装の現場を生み、実装が新しい研究課題を生み、研究が教育を更新し、教育が次の発信を生む——この循環の速度と質を最大化することが、本研究所の最も重要な責務です。

v2 を正本とする以上、原案と v0.5 の関係を以下のいずれかで決着する必要あり：

- **A案**: manifesto.md を v2 準拠で書き直す（原案破棄）
- **B案**: manifesto.md を「原案アーカイブ」として残し、v2 を新マスターとして別ファイル化（例: `manifesto-v2.md`）
- **C案**: 両者を併存させ、対外表現は v2、内部判断は原案を保持（複雑化リスク高）

→ 推奨は B案。原案の思想（資本分散・問いの品種改良場・第三の場所）は v0.5 では使わないが、第二弾以降の意思決定材料として記録に残す価値が高いため。
