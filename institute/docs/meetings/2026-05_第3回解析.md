## 1. 何が確定したか（why-what.md 比で前進した点）

### A. ラボ＝「サーバーラック / 業界団体インキュベーション」モデル（今回の最大の論点）

why-what.md で曖昧だった「場の機能」が、明確な比喩で言語化された：

| 役割 | 内容 |
| --- | --- |
| デジハリラボ自体 | **インフラ**（サーバーラック、コワーキング、事務局機能、法人化コスト肩代わり） |
| 内側で動くもの | **分科会＝独立した活動団体**（PPP、シンギュラボなど）が「ガチャッ」と差し込まれる |
| 課金イメージ | 一般社団法人化に年40-50万かかるところを、月1-10万でデジハリラボに乗れる |
| 比喩 | 「0050（フリーダイヤル）」「アイブリー」「AWSのコンソーシアム版」 |

→ why-what.md の「資本分散型」「二層構造」が、より具体的なオペレーションモデルに着地。
→ **この構造は外には出さない**（後述「外向き方針」参照）。

### B. 外向きメッセージは「研究を民主化する」一本に絞る

二択議論の結果：

- **「研究を民主化する」 ← 採用**（Chiaki が直感的に強く引かれた）
- 「新しい産業を作る」 ← 内部の動機としては保持、外には出さない

**訴求の二層ルール:**
> 外には**富士山（研究の中身）だけ見せる**。サーバーラック（インフラ）は見せない。「富士山見てる人は大太良（インフラ）は見えてない、それでいい」。

### C. ターゲットの絞り込み — 「袋閉じ」の人を狙う（Chiaki の新コンセプト）

- 世の中で「袋が閉じてる＝まだラベルが貼られていない・得体が知れない」段階のものをエンパワーする
- 例: Discord と Slack の文化差を嗅ぎ分けられる人、ロブロックスやマインクラフトの異能、「在野の研究家」
- 「Fill the World of Yours」 — 自分の世界を満たすことを既に知ってる人から知恵を借りる
- 藤井の「在野の研究家」と地続きで、対象像がより具体化

### D. 「やらないこと」の追加

why-what.md の表に追加すべき項目：

- **「うぞうむぞ」の大量集客はしない**（俗物図鑑の議論。1,300人集めても異能は100人いれば多い方）
- **中身の運営はしない**（インフラ提供に徹する）

## 4. why-what.md からの再確認 / 補強

- **シンギュラボとの提携意義** — ナショナルセキュリティの「攻める研究」が国立大ではできない、デジハリ（株式会社立）ならできる、というドメイン特定の優位
- **デジハリ固有アセット**の解像度向上：
  - OB/OG **5,000〜7,000人**
  - 元教員コミュニティ **500人**
  - 全国 **50〜60** のデジハリスタジオ（地方フランチャイズ）
- **「お墨付き」の価値** — デジハリラボ公認＝今治タオル方式。準教授→教授で人は喜ぶ、本源的欲求。B Corp / IU の教授バッジが「価値にならなかった」失敗例との差。

## 5. 構造的に見えてきた論点（why-what.md と接続）

why-what.md の **「問いの品種改良場」** ポジションは、今回の **「サーバーラック」** モデルと**同じものを別角度から言っている**：

- 問いの寿命 6-18ヶ月で淘汰 = サーバーラックに分科会を抜き差しする
- 答えの工場ではない = 中身は持たない、インフラだけ持つ

→ 次の文書で **「問いの品種改良場 ＝ 業界団体インキュベーション」** として統合するのが自然。
