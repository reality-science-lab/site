# デジハリラボ — WHY / WHAT

> 2026年4月17日の準備室ミーティングを中心に、ラボの方向性をWHY（なぜやるのか）とWHAT（何をやるのか）に整理した。

---

## WHY — なぜやるのか

### 1. アカデミアは構造的に「社会に届けられない」

研究者100人中99人は誰にも知られずに終わる。ムーンショット級の大型助成でも「誰一人何一つできていない」。JSTや学校法人の構造上、研究者が社会に価値を届ける会社を作ろうとすると「やめろ」と言われる。**研究と社会の間にパイプラインが存在しない。**

### 2. 「面白いからやる」では足りない——藤井直敬の原体験

研究者時代に10年で20〜25億円を使い、自分で会社（ハコスコ）を始めて初めてその重みに気づいた。知的好奇心と社会への還元衝動は混在しているが、「面白いからやる」だけでは正当化できない。意味と価値に向き合う覚悟。資本主義の経済合理性と真剣に向き合ったアカデミア出身者は極めて稀で、藤井さんはその数少ない一人。

### 3. 単一指標による評価の時代が終わる

KPI・偏差値・MBTIのような単純化は「人間の処理能力の限界が強いてきた歴史的妥協」だった。AIの登場でそのストッパーが外れた今、人間の複雑性をそのまま扱える時代が来ている。ラボはこの転換点に立つ。

### 4. 伊知哉さんの「オープンなメディアラボ」の未完のビジョン

KMDもIUも、伊知哉さんが目指したメディアラボにはならなかった。IUを辞めた今、コア教授約20人の受け皿と、構想を実現する新しい器が必要。

### 5. 今しかできない——3人の合流と時限性

杉山知之・藤井直敬・中村伊知哉。3人とも1990年代にMITで「こういう場が日本に必要だ」と感じ、30年間別々の道（教育・科学・政策）を歩いた末に、初めて合流できる条件が揃った。

ただし藤井さんは研究者＋起業家であり、理研を離れ、ハコスコをExitし、常に次へ移ってきた。**藤井がいる間に形にしなければ消える、時限付きの構想。**

---

## WHAT — 何をやるのか

### 1. 資本分散型のラボモデル

従来の「リターンが中枢に帰ってくる」モデルを明確に捨てる。


| 要素   | 仕組み                                |
| ---- | ---------------------------------- |
| 売上   | 各自のホーム（自社）で立てる                     |
| シード  | デジハリ経由の産官学連携助成金を活用                 |
| 還元   | 助成金が入った場合、デジハリは管理費のみ取り、残りを各社・各人に配分 |
| 新規事業 | 助成金をシードとして、このやり方でしかできない種類の事業を起こす   |


「このやり方でしかできない種類の新規事業の起こし方がある」——これが藤井さんの核心的な仮説。

### 2. 二層構造の振る舞いを持つ「場」

- **内側（信念）**: ラボ自体が「きちんと定義できないものとして始まる」。藤井さんを研究者・起業家・アーティストのどれとも呼べないように、ラベリングしないことを前提とする
- **外側（社会実装）**: 一方で「在野の科学者・研究家」のような、今まで存在しなかった新ジャンルを社会に向けて定義する

**内側は定義しないことを信じ、外側には分かりやすい新ジャンルを作る。** この二層構造がラボのコンセプト。

### 3. 化学反応を起こす「場」

伊知哉さんの強みは「三越」的な求心力——説明コストが低く、異業種の立場ある人が無償で集まってくる場を作る力。藤井さんが「汗をかいて耕す」開拓者で、太田さんが「その場所で楽しめることをやる」実行者。伊知哉さんはブランドとして居続ける。

この**役割の非対称性**がエンジンになる。

### 4. 人間の複雑性をそのまま扱う研究群

ラボの中で走る具体的な研究テーマとして:

- **ニューロダイバーシティの科学的汎用化** — 脳の特性パターンから「強い脳の組み合わせによる成功の汎用技」を実証する（藤井）
- **現実科学（Reality Science）** — 現実そのものが設計可能であるという仮説の検証（藤井）
- **複雑性を削ぎ落とさないアセスメント** — MBTIのように単純化するのではなく、人間の複雑性をそのまま全部拾うことで単一指標社会からの脱却を目指す（加藤）

### 5. 「問いの品種改良場」としてのポジション

デジハリの本当のDNAは「適応速度」——カリキュラムを年1-2回更新し、CG→Web→モバイル→XR→AIの各転換点で他校より先に動いてきた。このDNAを活かすなら、問いを固定して深く掘る従来のラボ（理研、MIT）ではなく、**問いの生成と淘汰を高速で回す機関**——まだ世界のどこにも存在しないポジション。

問いの寿命は6-18ヶ月。検証して筋が良ければスピンアウト、悪ければ殺す。答えの生産工場ではなく、問いの品種改良場。

## 構造的に見えてくること

このラボの本質は**「アカデミアと社会の間に、第三の場所を作る」**こと。

- 大学でもない、企業でもない
- 資本が中枢に集まるのでもない、各自が外で稼ぐ
- ラベルで定義するのでもない、複雑性をそのまま保つ
- 既存の評価軸で測るのでもない、新しい評価の仕組みを作る

「在野の研究者が、助成金をシードに、社会に届く事業を起こす」——この一文に圧縮できるが、そこに至る動機の層が深い。

## やらないこと


| やらないこと              | 理由                                     |
| ------------------- | -------------------------------------- |
| 「MITメディアラボを超える」と掲げる | 実績なき段階では大言壮語                           |
| コンテンツ産業のR&D部門を目指す   | 大手は自前で持っている。中小が欲しいのは研究知見ではなく制作費と採用チャネル |
| 収益還流サイクルを初期前提に置く    | 世界13事例を調査しても完全実現はゼロ。10-20年後のゴール        |
| 「デジハリ」の名称をそのまま使う    | 社内でネガティブな反応あり。別名での立ち上げも選択肢             |
